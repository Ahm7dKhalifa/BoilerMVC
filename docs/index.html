BoilerMVC

... is an ASP.NET MVC 3 boilerplate web application template aimed at newbies and advanced users. The template implements a variety of patterns and practices, and utilizes a handful of NuGet packages.

BoilerMVC is released under the Apache License v2.0 allowing it's usage in any scenario (personal, commercial, non-profit etc.).

Architecture

N-tier

BoilerMVC is separated into 6 different projects:

BoilerMVC.Common
Contains functionality used accross all layers: extension methods, interfaces etc.
BoilerMVC.Data
Functions as the data access layer housing Entity Framework classes and repositories.
BoilerMVC.Framework
Contains components and classes used by ASP.NET MVC: view models, HTML helpers, base controller classe.
BoilerMVC.Services
Contains business logic classes. Acts as intermediate between the database and the web layers.
BoilerMVC.Tests
Contains unit tests.
BoilerMVC.Web
ASP.NET MVC application project. Contains views, controllers and all website related content.
